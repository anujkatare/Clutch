We absolutely love community contributions! Here is how you can help shape Clutch:

Find an Issue: Head over to our GitHub Issues and look for tags like good first issue or help wanted.

Fork the Repo: Click the "Fork" button on the top right of this page.

Create a Branch: Create a branch for your feature or bug fix:


=>  git checkout -b feature/your-awesome-feature

Commit and Push: Write clean code, verify it compiles inside the Docker container, and push your branch:


=>  git commit -m "feat: Add support for custom feature"
=>  git push origin feature/your-awesome-feature

Open a Pull Request: Submit a Pull Request (PR) to our main branch. Our automated tests will run instantly to verify your changes!
